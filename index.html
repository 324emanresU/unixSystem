UNIX Overview

  • First OS developed in C language (1973).
  • Created by Ken Thompson and Dennis Ritchie at AT&T Bell Labs (1969).
  • Major versions: BSD, System V.
  • Linux: UNIX-like open-source OS by Linus Torvalds (1991).
  • Features: Interactive, multi-user, multitasking, portable, hierarchical file system.
  • Structure:
    • Kernel: Manages resources and processes.
    • Shell: User interface for commands.
    • Utilities: Development and editing tools.

UNIX System Access & Commands

  • Access: Connect via SSH (ssh userID@serverIP) and password.
  • Basic Editing:
    • Backspace: delete character
    • Ctrl+W: delete word
    • Ctrl+U: delete line
  • Command Format: command [options] [arguments] (e.g., ls -l /tmp)
  • Useful Commands: banner, date, clear, man
  • Change Password: passwd
  • Logout: logout, exit, or Ctrl+D
